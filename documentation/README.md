# Documentación de la ontología

Esta carpeta contiene toda la **documentación orientada a humanos**.  

# Propósito
El objetivo de este directorio es almacenar la **documentación de la ontología orientada a humanos** para ser publicada en línea y facilitar la comprensión de la conceptualización y su uso por terceros.

Esta documentación facilita la **reutilización y mantenibilidad** de la ontología.

# Contenido
Incluye aquí cualquier archivo utilizado para publicar la ontología, como:

- **Archivos HTML** — Generados por [Widoco](https://github.com/dgarijo/Widoco). Estos archivos también pueden incluir ejemplos, consultas SPARQL, enlaces a formas SHACL, etc.

# Formatos aceptados
Puedes utilizar formatos como:
- `.html` — Archivos HTML  
- `.md`, `.pdf` — Informes o documentación de resultados de pruebas  
- archivos adicionales necesarios para el contenido HTML.

# Mejores prácticas
- Mantén los archivos de documentación **versionados** junto con las actualizaciones de la ontología.
